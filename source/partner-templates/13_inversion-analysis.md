# Inversion analysis — how to LOSE this bid (so we don't)

> "Invert, always invert. … A lot of success comes from avoiding the standard ways of failing."
> — Charlie Munger

Instead of asking "how do we win N.Y.C. Groceries?", we ask the inverted question: **"How would we
guarantee we lose it?"** — then systematically refuse to do those things. Most RFPs aren't lost on
talent; they're lost on avoidable, self-inflicted errors. This is the list of ways to lose, grouped
worst-first, each flipped into a rule.

## Tier 1 — Instant losses (thrown out before anyone scores the work)

These don't cost points; they end the bid. Munger's lesson: **the catastrophic, irreversible errors
matter far more than marginal gains.** Eliminate every one of these first.

| How to lose | The rule (do the opposite) |
|---|---|
| Submit after **June 30, 4:00 PM ET** | Submit the day before; never trust a same-hour upload |
| Wrong format — not a searchable PDF, not a single zip, mislabeled | Single zip, searchable PDF, labeled `…RFP Response – DoChakki` |
| Omit a required appendix (Statement of Agreement, M/WBE Narrative, Participation Plan, Doing Business Form, Appendix 6) | Use the checklist; verify every appendix is in the zip |
| **Pitch brand concepts** (explicitly forbidden) | Show *approach and capability* only — never speculative logos |
| Exceed the **30-page** narrative cap | Budget pages; confirm the count before zipping |
| Unsigned Statement of Agreement / missing signatory | Get an authorized principal's signature early |
| Submit via email or any non-RFP-website channel | Upload only through edc.nyc/rfps |

## Tier 2 — M/WBE failure (loses the 15% and risks "non-responsive")

| How to lose | The rule |
|---|---|
| Count an **out-of-state / national WBE** (e.g. RSM's Texas cert) toward 30% | Count **only NYC-SBS-verified** firms; verify number + expiry |
| M/WBE participation lands **below 30%** | Build to ≥30% of contract value; add JPD if short |
| Appendix 4 award amounts **don't match** Appendix 6 | Reconcile the two documents before submitting — reviewers cross-check |
| Token M/WBE roles that look like box-checking | Give certified partners **core** scope (Two Twelve on Tasks 2B/3) |
| Name a firm that won't sign the Intent-to-Perform | Confirm signature willingness before listing anyone |

## Tier 3 — Throwing away the 75% (experience + proposal quality)

| How to lose | The rule |
|---|---|
| Generic portfolio with no **brand-→-built-environment** proof | Lead with case studies that show identity *and* its physical extension |
| No **civic / food-access** understanding; treat it like a retail launch | Frame the brand as public trust + affordability across 5 boroughs |
| No evidence of **A&E collaboration** (the hard 2027–2029 phase) | Foreground Two Twelve's architecture-team pedigree |
| Vague, taskless "approach" | Address every task (1A–4B) with a concrete method + schedule |
| Thin or padded bios; irrelevant work samples | Curate ruthlessly to *similar size and scope* |

## Tier 4 — Process & strategic self-sabotage

| How to lose | The rule |
|---|---|
| Miss the **June 12** question window | Send the 8 questions on time — it's the only clarification shot |
| Ignore NYCEDC's **June 18** posted answers | Read them; they can change the 30% base and Task 3 scope |
| Skip checking the RFP site for addenda | The website governs; check it before submitting |
| Compete on **price** (only 5% of score) | Be credible, not cheapest; win on experience + quality |
| Include **sales tax** in pricing | NYCEDC is tax-exempt — exclude it |
| Lowball Task 3 so it can't be sustained 2027–2029 | Price against a stated review-cycle assumption |
| Look like every other brand agency | Hammer the one differentiator: brand that becomes architecture |

## The Munger short-list (if you remember only this)

The bid is won by **not losing**. In priority order, never let these happen:

1. **Miss a deadline or botch the file format** — instant, total loss for zero reason.
2. **M/WBE under 30% or counting an uncertified firm** — fails a scored gate and risks DQ.
3. **Thin experience evidence** — forfeits the single biggest score (40%).
4. **No NYC-specific, civic understanding** — forfeits much of the 35%.

Avoid those four and DoChakki is already ahead of most respondents — *before* the work's quality is
even judged. Everything in `09_win-strategy.md` then turns "not losing" into "winning."
